### How `poll.js` works
1. **Reads `PREMBLY_API_KEY`** from the environment — exits with an error if missing.
2. **GETs** `https://api.prembly.com/api/v1/checker-widget/sdk/sessions/` with the `x-api-key` header.
3. **Counts sessions** by inspecting the response:
   - If the root value is an array → uses its length.
   - If the root value is an object → finds the first array-valued property and uses its length (handles the common `{ status, detail: [...] }` shape).
4. **Writes** `Total sessions: <count>` to `/home/user/app/output.log`.
5. **Logs** progress and errors to stdout; exits with code `1` on failure.
