### How it works

1. **`fastify-raw-body`** is registered with `global: false` and `encoding: false` so the raw `Buffer` of the request body is available on `request.rawBody` (opt-in per route via `{ config: { rawBody: true } }`).
2. For every `POST /webhook` request the handler:
   - Returns `401 {"error": "Invalid signature"}` if the `x-prembly-signature` header is absent.
   - Computes `HMAC-SHA256(rawBody, PREMBLY_API_KEY)` and Base64-encodes it.
   - Uses **`crypto.timingSafeEqual`** to compare signatures, preventing timing attacks.
   - Returns `401` on mismatch, or `200 {"status": "received"}` on success.
