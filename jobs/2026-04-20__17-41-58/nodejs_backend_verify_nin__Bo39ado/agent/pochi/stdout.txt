**`verify.js`** exports a single `verifyNIN(ninNumber)` function that:

- Makes a **POST** request to `https://api.prembly.com/verification/vnin`
- Sends `{ number: ninNumber }` as the JSON body
- Reads `PREMBLY_APP_ID` and `PREMBLY_API_KEY` from environment variables and passes them as `app-id` and `x-api-key` headers respectively
- Returns `response.data` from the Axios response

**Usage example:**
```js
const { verifyNIN } = require('./verify');

// Set env vars before calling:
// process.env.PREMBLY_APP_ID = 'your-app-id';
// process.env.PREMBLY_API_KEY = 'your-api-key';

verifyNIN('12345678901')
  .then(data => console.log(data))
  .catch(err => console.error(err));
```
