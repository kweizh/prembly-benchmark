# Prembly Webhook Signature Verification ## Background Create an Express.js server that receives Prembly webhooks and securely verifies their signature. Prembly sends webhooks with an `x-prembly-signature` header containing a Base64 encoded HMAC-SHA256 hash of the request body, using your API key as the secret.
