### Implementation Details:
- **Project Location**: `/home/user/nin-cache`
- **Main File**: `index.js`
- **Dependencies**: `express`, `axios`
- **Endpoint**: `POST /api/verify-nin`
- **Port**: 3000
- **Caching Strategy**: An in-memory `Map` is used to store verification results keyed by the `nin`. If a `nin` has been verified previously, the cached result is returned immediately, reducing Prembly API calls and improving performance.
- **Prembly Integration**: Connects to the sandbox API at `https://api.prembly.com/verification/nin` using the `PREMBLY_APP_ID` and `PREMBLY_API_KEY` environment variables.
